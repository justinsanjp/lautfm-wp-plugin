Ein Custom Radio Player für laut.fm mit Live/Playlist Mode setting

== Description ==
Dieser Player ermöglicht das Abspielen von laut.fm Radio Streams mit Play/Pause Funktion, 
Lautstärkeregelung und Anzeige der aktuellen Titel.
um den sender zu wächseln lade das script von Github, und änder den sender namen manuel

== Installation ==
Wenn Embeded
1. Lade den Plugin-Ordner in das `/wp-content/plugins/` Verzeichnis hoch
2. Aktiviere das Plugin im WordPress Admin-Bereich
3. Nutze den Shortcode [lautfm_radio] um den Player einzubinden

Wichtig:
Nach der aktivirung, gehe in die einstellungen und wächsel auf Widget um die Laut.fm richtlininen einzuhalten. die verwendung von Embeded ist auf eigener gefahr, ich übernehme keine haftung.
Weitere infos zur beschränkungen findet ihr im laut.fm forum:
https://forum.laut.fm/
